# Final Mechanical Structure

This folder contains the **final validated CAD models** of the vehicle’s mechanical architecture.  
These files represent the approved structural configuration intended for manufacturing, integration, and full system validation.

This release defines the stable mechanical baseline of the project.

---

## Folder Overview

- **assembly** — Complete mechanical assemblies ready for integration and validation.
- **full_vehicle** — High-level CAD model of the entire vehicle structure.
- **parts** — Individual finalized mechanical components used in the assemblies.

---

## Design Intent

The final structure was developed considering:

- Structural stability under expected operational loads  
- Modular integration with electronics and firmware  
- Manufacturability and assembly feasibility  
- Accessibility for maintenance and testing  
- Mechanical alignment with control system requirements  

---

## Release Status

This directory represents the **approved mechanical release**.  
Experimental models and intermediate iterations are stored outside this folder.

Any modification to the structure must be version-controlled and documented.
